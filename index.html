Modeling percolation system that is an nxn grid that represents each grid as open, closed, or filled. A system percolates if any of the bottom row is filled. To fill the grid you start by filling any 1xn spot that isn't close and the surrounding neighbors of that spot is filled if it can be filled. This effect happens until no more spots can be filled.
A wordnet is a database of words that have semantic relation including synonyms, hyponyms, and meronyms. In this project you create a wordnet digraph that stores vertex v as synonyms or synsets that have a directed edge to w such that w is a hyponym of v. This project also includes finding the shortest ancestral path between two vertices.
